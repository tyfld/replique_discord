# Discord Replique

Une appli clone de Discord créée avec Electron

## Installation

### Etape 1: Cloner le  project

```
git clone <url_repo>
cd replique_discord
```

### Etape 2: Installer les dépendances

```
npm install
```

### Etape 3: Lancer l'application

```
npm start
```


### Tray icon not visible

Attention, l'icône du Tray est invisible.
